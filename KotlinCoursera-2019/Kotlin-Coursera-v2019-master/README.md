# Котлин как первый язык программирования

Двенадцать групп задач (уроков).

1. Простые функции.
2. Ветвления.
3. Циклы.
4. Работа со списками.
5. Работа с ассоциативными массивами (Maps).
6. Разбор строк.
7. Работа с файлами.
8. Простые классы.
9. Сложные классы на примере матриц.
